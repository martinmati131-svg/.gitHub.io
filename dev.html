2. ROS 2 Node Architecture

The system operates using a modular node structure to ensure low-latency response to field telemetry.

# Launching the Bio-Extraction Node
ros2 run aura_industrial extractor_controller_node --ros-args -p batch_mode:=synergy_extraction

Current active topics include /aura/soil/ph, /aura/industrial/yield_est, and /aura/environment/humidity.

3. Industrial Logic (Decision Matrix)

The "Aura Brain" uses a conditional logic gate to determine extraction potency and foliar feed advisories.

def evaluate_tithonia_synergy(ph, humidity):
    if ph < 5.8:
        return "PH_SOLUBILIZATION_ALERT"
    if humidity > 75:
        return "ONION_ANTIFUNGAL_BOOST" # Synergistic additive
    return "STABLE_FERMENTATION"
